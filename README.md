# 🛠️ SUT FixIt - ระบบแจ้งซ่อมแซมหอพักนักศึกษา มทส. (Comprehensive Guide)

**SUT FixIt** คือแพลตฟอร์มแอปพลิเคชันบนมือถือ (Mobile Application) ที่ถูกออกแบบมาเพื่อยกระดับและแก้ปัญหาความล่าช้าในการแจ้งซ่อมแซมสิ่งอำนวยความสะดวกภายในหอพักนักศึกษา มหาวิทยาลัยเทคโนโลยีสุรนารี (SUT) โดยเปลี่ยนจากการเขียนกระดาษแจ้งซ่อมแบบเดิมๆ มาสู่ระบบดิจิทัลแบบ Real-time เต็มรูปแบบ

---

## 🌟 ฟีเจอร์หลักแบ่งตามผู้ใช้งาน (Features by Role)

ระบบถูกออกแบบสถาปัตยกรรมมาเพื่อรองรับผู้ใช้งาน 3 กลุ่มหลักอย่างเป็นระบบ:

### 🧑‍🎓 1. นักศึกษา / ผู้พักอาศัย (Resident)
* **Smart Reporting:** แจ้งซ่อมแซมได้ง่ายๆ โดยระบุหอพัก เลขห้อง และหมวดหมู่ปัญหา (เช่น ประปา, ไฟฟ้า, เครื่องใช้ไฟฟ้า)
* **Rich Media Upload:** รองรับการแนบภาพประกอบปัญหา โดยสามารถเลือก **"อัปโหลดจากคลังภาพ (Gallery)"** หรือ **"ถ่ายรูปใหม่จากกล้อง (Camera)"** ได้ทันที
* **Real-time Tracking:** ติดตามสถานะงานซ่อมของตัวเองได้แบบเรียลไทม์ (รอรับงาน -> กำลังดำเนินการ -> รอตรวจสอบ -> เสร็จสิ้น)
* **Profile Management:** จัดการข้อมูลส่วนตัวและแก้ไขรูปโปรไฟล์ประจำตัวได้ (เชื่อมต่อกับ Firebase Storage โดยตรง)

### 👷‍♂️ 2. ช่างเทคนิค (Technician)
* **Task Board:** กระดานแสดงรายการแจ้งซ่อมที่รอการดำเนินการ ช่างสามารถกดรับงานที่ตรงกับสายงานตัวเองได้
* **Location Tracking:** ระบบแสดงพิกัดจุดเกิดเหตุบนแผนที่ (GPS) ช่วยให้ช่างเดินทางไปยังหอพักและห้องที่เกิดเหตุได้แม่นยำ
* **Job Closing System:** ระบบปิดงานซ่อมที่สมบูรณ์แบบ ช่างต้องบังคับกรอก "รายละเอียดการซ่อม", "ค่าวัสดุอุปกรณ์", และที่สำคัญคือ **"ถ่ายรูปผลงานหลังซ่อมเสร็จ"** เพื่อใช้เป็นหลักฐาน
* **Push Notifications:** รับการแจ้งเตือนทันทีเมื่อมีนักศึกษาแจ้งซ่อมใหม่

### 👨‍💻 3. ผู้ดูแลระบบ (Admin)
* **Admin Dashboard:** กระดานตรวจสอบภาพรวมของงานทั้งหมด สามารถใช้ Filter กรองดูงานตามสถานะ (รอตรวจงาน, กำลังซ่อม, เสร็จสิ้น)
* **Quality Assurance (QA) Review:** ก่อนที่งานจะถูกปิดสมบูรณ์ แอดมินสามารถเปิดดูข้อมูลเปรียบเทียบ **"รูปภาพก่อนซ่อม"** และ **"รูปภาพหลังซ่อม"** พร้อมดูค่าใช้จ่ายที่ช่างเบิก
* **Final Approval:** ระบบอนุมัติปิดงาน เมื่อแอดมินตรวจสอบแล้วว่างานเรียบร้อย สามารถกดปุ่มอนุมัติ ระบบจะเปลี่ยนสถานะเป็น "เสร็จสิ้น" และส่งแจ้งเตือนกลับไปยังนักศึกษา

## 💻 เทคโนโลยีและไลบรารีที่ใช้ (Tech Stack & Dependencies)

โปรเจกต์นี้พัฒนาด้วยเทคโนโลยีสมัยใหม่ เพื่อประสิทธิภาพและความเสถียรสูงสุด:

* **Framework:** React Native, Expo (`expo-router` สำหรับการทำ File-based routing)
* **Language:** TypeScript / JavaScript (`.tsx`)
* **Backend as a Service (BaaS):** Firebase
  * `Firebase Auth` - จัดการระบบ Login / Register
  * `Firestore Database` - ฐานข้อมูล NoSQL แบบ Real-time
  * `Firebase Storage` - จัดการไฟล์รูปภาพ (รูปแจ้งซ่อม, รูปโปรไฟล์)
* **Core Libraries:**
  * `expo-image-picker` - จัดการการเข้าถึงกล้องและอัลบั้มรูปภาพ
  * `expo-file-system` - จัดการไฟล์ชั่วคราวก่อนอัปโหลด
  * `react-native-maps` - แสดงแผนที่และพิกัดพอยน์เตอร์
  * `@expo/vector-icons` - จัดการระบบ UI Icons
